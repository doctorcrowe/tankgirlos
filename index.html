// HELP
01 // GET AN API KEY
This app uses Fireworks AI to power the chat. It's free to sign up and you get credits to start.

→ Sign up at fireworks.ai

Once you have an account, go to API Keys and create a new key. It looks like fw_xxxxxxxxxxxxxxxx.
02 // ADD YOUR KEY
Click ⚙ in the top right corner to open Settings. Paste your API key into the FIREWORKS API KEY field and hit APPLY.

Your key is saved in your browser only — it never leaves your device.
03 // START CHATTING
Type anything in the box at the bottom and press Enter or Execute. Ask it questions, get help writing something, brainstorm ideas — anything you'd use ChatGPT for.

Adjust the MOOD slider in Settings to change the personality — from unhinged chaos to strictly professional.

In Settings you can:

> Change the character name and bio
> Upload your own avatar image (click the character on the right)
> Export your setup as a .skin file to save or share
> Import a skin someone else made
> Copy a share link to send your character to anyone
